.Net Client

The .NET client can be downloaded as a binary distribution or a source distribution.

.NET Client 0.4
.NET Client Source 0.4

To install the client, create a directory and extract the contents of the zip file into that directory.

The client runs as a windows application. All configuration should be done through the client. Do not directly edit the XML file!

Setup the configuration, save, and press the 'Start' button.
