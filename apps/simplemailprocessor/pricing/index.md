## Development Version
{:.subheader #development}

A limited use development version of Simple Mail Processor is available for [immediate download](../download). This version provides the full Simple Mail Processor functionality but is limited to 100 emails per execution. After 100 emails, this version will stop processing new emails until the application is restarted.
