# Applications

## [Playlist Export](/apps/playlistexport)
{:.section-header}

[Playlist Export](/apps/playlistexport) is a Mac OSX desktop application that
enables users to export playlists defines in Apple's iTunes application
to playlist formats, including M3U, WPL, ZPL, and Centrafuse (MPL).
This application simplifies listening to your music on 3rd party devices while still using iTunes as your primary music catalog.

## [Simple Mail Processor](/apps/simplemailprocessor)
{:.section-header}

[Simple Mail Processor](/apps/simplemailprocessor) is a Java library that allows developers to quickly and easily add the ability to process incoming emails to an application.
