Applications

Playlist Export

Playlist Export is a Mac OSX desktop application that enables users to export playlists defines in Apple's iTunes application to playlist formats, including M3U, WPL, ZPL, and Centrafuse (MPL). This application simplifies listening to your music on 3rd party devices while still using iTunes as your primary music catalog.

Simple Mail Processor

Simple Mail Processor is a Java library that allows developers to quickly and easily add the ability to process incoming emails to an application.
